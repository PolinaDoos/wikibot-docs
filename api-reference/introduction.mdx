## Основы

Базовый URL: https://api.tomleto.pro/.

Взаимодействие с API осуществляется при помощи HTTP POST и HTTP GET запросов. Убедитесь что POST запрос содержит заголовок Content-Type: application/json.

## Асинхронный режим

Поддерживаются синхронный и асинхронный механизмы ответа. Все запросы являются синхронными по умолчанию. Для перехода на асинхронный механизм необходимо задать webhook URL.

Синхронные запросы возвращают ответ, либо ошибку в контексте текущего HTTP-запроса.

Асинхронные запросы сразу же возвращают пустой HTTP 200 ответ и вызывают webhook URL с результатами выполнения запроса.

## Авторизация

Все запросы должны содержать заголовок Authorization

```
Authorization: my_api_key
```
