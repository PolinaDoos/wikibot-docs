Данная интеграция возможна только при наличии платной версии Bitrix24.

Инструкция в два этапа, на первом шаге будет происходить создание и настройка бота, на втором шаге будет подключение созданного бота к Открытой Линии.

## Создание и настройка бота

- Перейти по меню Приложения -> Разработчикам -> Добавить чат-бот и выбрать пункт Передавать боту сообщения из чата
- В открывшемся окне необходимо произвести создание и настройку нового чат-бота.
- Скопировать текст из поля Вебхук для вызова rest api и передать его в WikiBot support.
- Ввести название бота, например “Мой бот”, URL обработчика бота https://api.tomleto.pro/webhook/bitrix24, тип бота Чат-бот для Открытых линий
- Пролистать ниже, включить исходящий вебхук и настроить его:
- Добавить URL обработчика https://api.tomleto.pro/webhook/bitrix24
- Передать Токен приложения в WikiBot support
- Добавить События – Новое сообщение из ОЛ, Новое сообщение боту
- Сохранить настройки бота

Далее подключаем созданного бота

## Подключение к Открытой Линии

- Перейти по меню CRM -> Интеграции -> Мессенджеры
- Выбрать Онлайн чат -> Создать открытую линию, если линия не была создана ранее
- Перейти в настройки Открытой Линии
- На вкладке чат-боты поставить галочку При обращении клиента назначить ответственным чат-бота и выбрать созданного бота из списка
- Настроить параметры бота исходя из требований. Для настройки Когда отключать чат бота выбрать значение После перевода на оператора
- Добавить виджет на сайт или интеграцию через Телеграм в меню CRM -> Интеграции -> Мессенджеры
- Проверить работу бота, написав сообщение в виджет или телеграм.

При интеграции для вашей компании могут быть добавлены любые специфичные опции, например: время работы ИИ-бота только ночью, логика переключения на оператора, типы вопросов на которые боту нельзя отвечать.
