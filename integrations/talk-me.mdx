## Автоматический режим (мастер)

Для создания бота перейдите по адресу https://app.tomleto.pro/bots/new.

Выберите имя для вашего бота и выберите интеграцию Talk-me.

Нажмите кнопку Create. Вы попадёте на экран просмотра API-ключа. API-ключ будет автоматически сгенерирован для вас. Вы можете его скопировать, нажав кнопку Copy.

Нажмите кнопку Next.

Вы попадёте на экран завершения настройки, после чего будете перенаправлены на страницу бота.

На странице бота вы можете выбрать пункт API Keys в меню и перейти на страницу с API-ключами, где можете просмотреть список и добавить новый ключ, либо скопировать один из существующих.

Для завершения интеграции, пожалуйста, перейдите на сайт Talk-me и следуйте предоставленным инструкциям.

## Ручной режим

Если вы создали бота не через веб-интерфейс, например через телеграм, то вы можете самостоятельно выпустить API-ключ в веб интерфейсе.

Для этого зайдите на портал Wikibot, войдите через телеграм, созданный через телеграм бот автоматически привяжется к вашему аккаунту.

Затем перейдите в раздел API Keys.

Нажмите кнопку Add API Key. В открывшемся диалоговом окне введите ключ, либо нажмите кнопку Generate, затем нажмите кнопку Save.

После добавления ключа наведите мышку на строчку таблицы с добавленным API-ключом и нажмите на иконку копирования для того, чтобы скопировать его.

Перейдите на сайт Talk-me и добавьте API-ключ в вашу интеграцию
