title: "L1"
description: "Первая линия поддержки для бота"
---

Ответ первой линии L1 Support – точные ответы без изменения формулировок в стиле FAQ.

Если срабатывает L1, то пользователь получает заранее готовый ответ или автоматическое переключение на оператора.

L1 – отлично подходят для тем где нужно отвечать строго по регламенту.

L1 – нужно использовать для переключения на оператора в темах где боту нельзя отвечать.

L1 – значительно дешевле чем ответ по базе знаний и может быть использован как альтернатива базе знаний. Подробнее о цене L1 https://wikibot.pro/pricing

L1 состоит из вопросов (Main question). Для каждого вопроса задается один из двух вариантов: переключение на оператора (Redirect to operator) или ответ (Answer)

Каждый Main question должен быть довольно узким, иначе он будет срабатывать очень часто.

Чтобы бот лучше понимал Main question для него необходимо другие формулировки вопроса (Similar questions). Не больше 10 Similar questions для каждого Main question.

Тестировать L1 Support удобно на портале в разделе Chat.
Если бот ответил “Хорошо, передал ваше обращение в работу — скоро вернёмся с ответом.” – значит сработало переключение на оператора в L1.
