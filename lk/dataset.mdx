---
title: "Источники данных"
description: "Источники данных для вашего бота"
---

В этом разделе осуществляется работа со всеми данными бота.

Данные делятся на следующие типы:

Ваша база знаний (сайт или документ Google Sheet):
кнопка Reload data source – запускает парсинг базы знаний
кнопка Edit data source – позволяет добавить/изменить базы знаний
Документы загруженные в систему (минуя базу знаний):
кнопка Add document – позволяет добавить документ
Индекс – данные на которых обучается бот. Можно не все данные из базы знаний и загруженных документов использовать для обучения бота
Add to index – добавление выбранных страниц в индекс
Remove from index – удаление выбранных страниц из индекса
Reload pages – перезагрузка по урлу выбранных страниц, т.е парсинг и переиндексация

Справа от каждой загруженной страницы есть кнопка Explore, нажав её можно добавить ключевые слова для страницы чтобы она лучше находилась.
