## Просмотр списка

В данном разделе вы можете просмотреть список созданных API ключей.

Для просмотра или копирования API ключа наведите указатель мыши на нужную строчку и нажмите соответствующую иконку.

<img src="/images/lk-api-keys.png" alt="api keys" />

## Создание ключа

Для создания нового API ключа нажмите кнопку **Добавить API ключ**. В открывшемся диалоговом окне введите ключ, либо нажмите кнопку **Сгенерировать** и затем нажмите кнопку **Сохранить**.

<img src="/images/integrations-verbox-add-api-key.jpeg" alt="verbox integration add api key" />
