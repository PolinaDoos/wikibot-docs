---
title: "API ключи"
description: "Создание, редактирование и просмотр и API ключей"
---

На странице вы видите список созданных API ключей. Для просмотра и копирования API ключа наведите указатель мыши на нужную строчку и нажмите соответствующую иконку.

Для создания нового API ключи нажмите кнопку Add key в правом верхнем углу.
