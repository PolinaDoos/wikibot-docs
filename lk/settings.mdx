На этой страние вы можете настроить основные параметры бота:
- Имя бота.
- Режим работы.
- Шаблоны сообщений.

## Имя бота
Введите имя бота в поле **Имя** и нажмите **Сохранить**.

<img src="/images/lk-settings-name.png" alt="settings bot name" />

## Режим работы
Определите рабочие часы и выходные дни, в соответствии с которыми будет работать ваш бот. В выходные дни бот будет использовать отдельные шаблоны сообщений.

<Info>
    Время местное.
</Info>

Для того, чтобы задать единое расписание на все дни, укажите интервал в поле **По умолчанию**.

Для того, чтобы задать особенное время для какого-то **конкретного** дня, переопределите время работы для этого дня.

Для того, чтобы отметить определённый день как выходной, нажмите галочку **Выходной**.

<img src="/images/lk-settings-work-hours.jpeg" alt="settings work hours" />

## Шаблоны сообщений

<Warning>
    Шаблоны сообщений содержат важные системные поля, такие как `{message}` и `{docs, docsFormatter}`. Убедитесь, что вы не удаляете их, либо делаете это сознательно, иначе бот не будет выдавать часть ответа, связанную с определённым полем. Также убедитесь что вы сможете восстановить эти поля, если понадобится их вернуть.
</Warning>

<Info>
    `{message}` — служебное поле для непосредственого ответа бота.

    `{docs, docsFormatter}` — служебное поля для списка документов по ответу бота.
</Info>

Задавайте шаблоны сообщений для различных сценариев.

Для применения изменений отредактируйте нужный шаблон и нажмите кнопку **Сохранить**. Обновленный шаблон сообщения применится в демо-чате справа.
