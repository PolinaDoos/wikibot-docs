Чтобы перейти к настройкам бота, кликните на карточку бота и выберите справа “Configure”.

Здесь вы можете кастомизировать, например:

приветствие;
сообщение бота, если он нашел ответ;
сообщение бота, если ответ не найден.

При ответе бота работает трехуровневый механизм:

1. Cначала срабатывают исключения, когда нужно переключаться на оператора. По умолчанию не заданы. Для настроек смотрите раздел Exclusions.

2. Ответ первой линии – точные ответы без изменения формулировок по FAQ-документу.

3. Ответы по базе знаний – бот ищет ответ в базе знаний и генерирует ответ.
